# v1.2.2 更新内容

## 新增
- 亚克力效果检测：自动检测系统是否支持 Win11 亚克力效果，不支持时自动回退到毛玻璃并显示警告
- 新增 `System.Drawing.Common` 依赖（用于视觉效果验证）

## 下载

下载 `ImageManager_Setup_1.2.2.exe` 并运行安装程序。

## 系统需求

- Windows 10/11 (x64)
- .NET 8 Runtime（包含在安装程序中）

## 官网

https://imagemanager-6gs.pages.dev/
