# 终端图片中转管理工具 v1.1.0

[English](#english) | [繁體中文](#繁體中文) | [日本語](#日本語) | [한국어](#한국어)

## 下载

下载下方并运行安装：`ImageManager_Setup_1.0.0.exe`

## 特色

- 拖放图片导入并带缩略图预览
- 截图用 Ctrl+V 剪贴板粘贴
- 磨砂玻璃背景，可调节透明度
- 多语言：zh-CN，zh-TW，英文，ko，ja
- 明亮/黑暗/系统主题
- 一键粘贴路径至目标窗口
- Inno Setup 安装程序，完全卸载

## 系统需求

- Windows 10/11（x64）
- .NET 8 运行时（包含在安装程序中）
