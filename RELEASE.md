# v1.2.4 更新内容

## 新增
- 自动识别剪贴板图片：开启后每500ms检测剪贴板，自动保存新图片
- 粘贴后回到目标窗口：粘贴完成自动回到之前的应用窗口
- 粘贴后删除文件：粘贴成功后自动删除源文件
- 设置页面拆分为独立章节：隐私、粘贴设置、通用设置

## 修复
- 修复设置窗口因缺失图标资源（clipboard.png、arrow-left.png）导致无法打开的问题
- 修复设置窗口中重复显示相同选项的问题
- 修复错误日志未复制AI修复提示词到剪贴板的问题
- 修复MainWindow毛玻璃背景和窗口置顶初始化

## 下载

## What's New in v1.2.4

### Added
- Auto-detect clipboard images: checks clipboard every 500ms and auto-saves new images
- Return to target window after paste: automatically returns to the previous app window
- Delete file after paste: auto-deletes source file after successful paste
- Settings page reorganized into independent sections: Privacy, Paste Settings, General Settings

### Fixed
- Fixed settings window crash caused by missing icon resources (clipboard.png, arrow-left.png)
- Fixed duplicate settings entries in the settings window
- Fixed AI fix prompt not being copied to clipboard on error
- Fixed MainWindow frosted glass background and always-on-top initialization

## v1.2.4 업데이트 내용

### 추가
- 클립보드 이미지 자동 감지: 500ms마다 클립보드를 확인하여 새 이미지를 자동 저장
- 붙여넣기 후 대상 창으로 돌아가기: 붙여넣기 완료 후 이전 앱으로 자동 돌아감
- 붙여넣기 후 파일 삭제: 붙여넣기 성공 후 소스 파일 자동 삭제
- 설정 페이지를 독립 섹션으로 분할: 개인정보 보호, 붙여넣기 설정, 일반 설정

### 수정
- 아이콘 리소스 부족(clipboard.png, arrow-left.png)으로 인한 설정 창 열기 오류 수정
- 설정 창에 동일한 옵션이 중복 표시되는 문제 수정
- 오류 시 AI 수정 프롬프트가 클립보드에 복사되지 않는 문제 수정
- MainWindow 뿌옇게 배경 및 항상 위에 표시 초기화 수정
