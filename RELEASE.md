# v1.2.3 更新内容

## 改进
- 「不再提醒新版本」移至行为设置卡片中，统一管理
- 开机自启图标更新
- 外观和字体设置图标更新
- 启动时后台检查尊重「不再提醒新版本」设置

## 修复
- 修复了 Win11 亚克力效果误判导致自动回退到毛玻璃的问题
- 修复了设置窗口动画效果不稳定的问题

## 下载

下载 `ImageManager_Setup_1.2.3.exe` 并运行安装程序。

## 系统需求

- Windows 10/11 (x64)
- .NET 8 Runtime（包含在安装程序中）
