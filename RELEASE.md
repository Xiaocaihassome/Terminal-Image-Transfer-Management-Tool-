# v1.2.1 更新内容

## 新增
- 粘贴后删除文件（设置中开启，粘贴到窗口时自动删除源文件）
- 开机自启（设置中开启，系统启动时自动运行）
- 不再提醒新版本（测试版，设置中开启，关闭更新检查和提醒）
- 错误日志功能（自动记录异常，支持一键复制 AI 修复提示词）
- 使用须知与免责声明
- 多文件粘贴支持（可同时粘贴多个文件路径）

## 改进
- 设置窗口布局优化（行为设置合并为一张卡片）
- 字体设置支持实时预览
- 所有开关默认关闭
- 检查更新按钮支持多语言

## 修复
- 修复了 Toast 提示文字显示不全的问题

## 下载

下载 `ImageManager_Setup_1.2.1.exe` 并运行安装程序。

## What's New in v1.2.1

### Added
- Delete file after paste (enable in settings, auto-deletes source file when pasting to window)
- Auto start on boot (enable in settings, launches automatically when Windows starts)
- Don't remind about new versions (Beta, enable in settings, disables update checks and notifications)
- Error logging (auto-records exceptions, supports one-click copy of AI repair prompt)
- Terms of use and disclaimer
- Multi-file paste support (paste multiple file paths at once)

- Settings layout optimized (behavior settings merged into one card)
- Font settings now support real-time preview
- All toggles default to off
- Check for updates button supports multiple languages
